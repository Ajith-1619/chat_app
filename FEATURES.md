# Watchtower Flow Features Brief

Date: 2026-07-14

## Core Messaging
- One-to-one chat, group chat, and channel conversations.
- Text messages with reply, forward, bookmark, pin, edit/delete direction, and message info.
- Read receipts, sent/read time, source device, and sender/read location metadata.
- Message search, global search, saved messages, archived channels, and chat folders.

## Attachments
- Send images, documents, PDFs, text files, CSV/XLSX, APK, HTML/PHP, and other file types.
- Multiple file upload support.
- Copy/paste and drag/drop file upload support on web.
- In-app file preview where supported.
- Download support for unrestricted files.
- Restricted-file direction for app-only viewing.

## Location
- Current location sharing as map-card message.
- Live location sharing with duration, update frequency, and stop sharing.
- Normal message send/read latitude-longitude stored as metadata only.
- Location address visibility controlled by permission.
- Message Info shows send/read address for allowed users.

## Voice, Contacts, and Checklists
- Voice message recording and sending.
- Contact sharing from mobile contacts.
- Live checklist messages with progress and checkbox updates.

## Groups and Channels
- Create groups/channels.
- Admin/owner member management.
- Add/remove members.
- Group/channel profile panel.
- Mentions and channel-specific workflows.

## My Hub
- Tasks & Tickets dashboard.
- Task list filters: open, closed, created by me, following, due today, overdue, stale.
- Task detail view with description, people, updates, and quick updates.
- Create task from chat messages.
- Update existing tasks from chat messages.
- Leave request screens.
- Attendance screens with punch in/out and calendar.

## Notifications
- System notification conversation.
- External notification API for OTP/system alerts.
- Push notification registration and dispatch.
- Wake-up notification support for groups/channels.

## Release Management
- Draft/live release tracking for Android, Web, Windows, and Linux.
- Version, build number, rollout, force update, and approval flow.
- Employee 302 approval path for moving draft builds live.

## Admin and Settings
- Profile and user detail screens.
- Appearance settings.
- Active sessions.
- Diagnostics dashboard.
- What’s New / release notes.
- Location visibility management.

## Developer / Governance
- Flow registry for requirements and implementation tracking.
- Audit/report requirement tracking.
- Work report documentation.
- UI design documentation in `ui.md`.
